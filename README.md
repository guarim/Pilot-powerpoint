# Pilot-powerpoint
Projet : En html et javascript Avec la bibliothèque média pipe et open cv détecter la main gauche et la main droite via la webcam, la page html doit piloter l’affichage d’images dans le dossier asset (1.png, 2.png ….) dans le même répertoire, 
si la main droite passe de la gauche vers la droite avec la main gauche poing fermé on passe à l’image suivante, si la main droite passe de la droite vers la gauche avec la main gauche poing fermé on passe à la l’image précédente, 
si seul l’index main droite est levé avec le poing mais gauche fermé on trace une ligne qui suit l’index, on efface le tracé si l'index touche le pouce de la main droite, si on ferme les deux poing on arrête le programme
